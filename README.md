# API Blog - INF222

Cette application est une API backend simple de gestion des articles de blog, réalisée avec **Node.js**, **Express**, **SQLite** et **Swagger**.

## Fonctionnalités

- Créer un article
- Lister tous les articles
- Filtrer les articles par catégorie, auteur ou date
- Lire un article par son ID
- Mettre à jour un article
- Supprimer un article
- Rechercher un article par texte dans le titre ou le contenu
- Documentation interactive avec Swagger UI

## Technologies utilisées

- Node.js
- Express.js
- SQLite
- Swagger UI
- express-validator
